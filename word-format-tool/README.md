# 智能论文&竞赛格式处理平台

## 项目介绍

智能论文&竞赛格式处理平台是一个专业、高效、智能的文档格式处理解决方案，专为大学生和研究人员设计，帮助他们快速处理论文和竞赛文档的格式问题。

### 核心功能

- **一键格式标准化**：支持多种竞赛和大学论文模板，一键应用标准格式
- **WPS自动生成导航**：绑定WPS标题样式，自动生成文档导航目录
- **知网参考文献优化**：标准化参考文献格式，解决查重标红问题
- **智能降重润色**：支持不同强度的润色，可通过AI模型实现深度润色
- **格式合规检查**：自动检查文档格式是否符合要求
- **竞赛规则检查**：检查文档是否符合各种竞赛的规则要求
- **文档处理前后对比**：直观展示处理效果

## 技术架构

## 快速开始

- Python 3.8+
- pip

```bash
pip install -r requirements.txt
```

### 运行应用

```bash

```

## 使用指南

### 1. 格式处理
1. 选择合适的模板
2. 上传待处理的Word文档
3. 配置处理选项（如绑定WPS样式、标准化参考文献）
4. 点击「开始格式处理」按钮
5. 查看处理结果和报告

### 2. AI润色降重
1. 上传待润色的文档
2. 上传查重报告（可选）
3. 选择润色强度和AI模型
4. 输入API Key（可选，启用AI深度润色）
5. 点击「开始AI润色降重」按钮
6. 查看润色结果和报告

### 3. 模板管理
1. 调整格式参数
2. 保存为自定义模板
3. 导出模板文件
4. 导入模板文件

## 技术文档

### 核心函数

#### process_doc()
- 功能：处理文档格式和内容
- 参数：文件对象、中文格式配置、英文格式配置等
- 返回值：处理后的文档、修改记录、标题统计等

#### rewrite_sentence()
- 功能：润色单个句子
- 参数：句子文本、润色配置、API Key等
- 返回值：润色后的句子、润色类型、语义保留度

#### competition_rule_checker()
- 功能：检查文档是否符合竞赛规则
- 参数：文档对象、竞赛类型
- 返回值：检查报告

### 配置文件

#### 模板配置
模板配置采用JSON格式，包含中文格式和英文格式设置，以及特殊要求。

#### API配置
使用AI润色功能需要配置相应的API Key。

## 常见问题

### 1. 文档处理失败
- 检查文件是否为有效的docx格式
- 检查文件大小是否超过限制（50MB）
- 检查网络连接是否正常

### 2. AI润色无效果
- 确保输入了正确的API Key
- 检查网络连接是否正常
- 尝试调整润色强度

### 3. 格式不符合要求
- 检查选择的模板是否正确
- 检查文档结构是否符合模板要求
- 尝试手动调整格式参数
