# 智排通 - 学术竞赛文档智能格式处理Web平台
2026年中国大学生计算机设计大赛 软件应用与开发类（Web应用开发赛道）参赛作品

## 📌 项目简介
本项目是面向大学生学术竞赛、毕业论文场景的一站式文档智能处理Web平台，针对学生群体「文档排版繁琐、赛事格式不熟悉、合规性扣分、查重降重难」的核心痛点，实现了**模板一键适配、文档格式标准化、AI智能润色降重、格式合规校验、参考文献规范优化**全流程闭环能力，无需用户掌握专业排版知识，一键生成符合赛事/高校要求的规范文档。

## ✨ 核心功能
1. **全场景模板库**：内置三创赛、互联网+、挑战杯等主流大学生竞赛模板，以及清华、北大、浙大等全国高校本科/硕士毕业论文模板，支持自定义模板导入
2. **一键格式标准化**：自动识别文档标题层级，精准修改字体、字号、行距、缩进、对齐方式等格式，无损修改不改动原文内容
3. **AI智能润色**：支持轻度/标准/深度三档润色模式，兼顾语义保留与降重效果，适配学术论文/竞赛文档的专业表述
4. **格式合规性自动校验**：处理完成后自动生成合规检查报告，标注格式违规项并给出修改建议
5. **多格式文档兼容**：支持.doc/.docx/.pdf格式文档的读取、转换与处理，适配用户全场景使用需求
6. **WPS导航自动生成**：自动绑定WPS标题样式，导出的文档可直接生成导航目录，无需手动调整

## 🛠️ 技术栈
- **后端核心**：Python 3.10+
- **Web框架**：Streamlit
- **文档处理**：python-docx、pdfplumber、textract
- **数据处理**：pandas
- **AI能力**：兼容豆包大模型API、即梦AI接口
- **工程化**：Git、Docker
